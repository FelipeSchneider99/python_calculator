# 🧮 Python Tkinter Calculator

A simple desktop calculator built with Python using the Tkinter library.  
This project provides a graphical user interface (GUI) for performing basic mathematical operations, along with additional features like square root and percentage calculations.

---

## 🚀 Features

- Basic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Square root calculation
- Percentage calculation
- Clear and backspace functionality
- Keyboard support (Enter and Backspace)
- Simple and responsive GUI

---

## 🖥️ Technologies Used

- Python 3
- Tkinter (built-in GUI library)

---

## 💡 How It Works

The user inputs numbers and operations through buttons or keyboard.

The expression is evaluated using Python’s built-in eval() function.

The result is displayed in the calculator screen.

Errors are handled and displayed as "ERROR".
